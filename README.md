Plastic Cleanup and Sales System

A full-stack web platform designed to organize eco-cleanup events and promote the sale of recycled plastic products. The system encourages community participation through gamification, real-time impact tracking, and a sustainable marketplace.

🚀 Features

🧹 Cleanup Event Management

Create and manage cleanup drives

Volunteer registration and participation tracking

🎯 Gamification

Reward users with points for participation

Leaderboard to encourage engagement

📊 Impact Tracking

Track plastic collected (in kg)

Visualize environmental impact in real-time

🛒 Marketplace

Buy and sell recycled plastic products

Promote sustainability and circular economy

👤 User Dashboard

View participation history

Track rewards and contributions

🛠️ Tech Stack

Frontend:

HTML5, CSS3, JavaScript

Backend:

Flask (Python)

Database:

MySQL
